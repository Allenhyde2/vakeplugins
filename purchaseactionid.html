데이터 병합 가이드

  1. 1

    구매 데이터 확보 (Purchase Items)

    허브모임 관리자 페이지의 주문 통합 검색 메뉴에서 조건을 설정하여 엑셀 파일을 다운로드합니다.

    파일명 예시: purchase_items.xlsx Purchase Items 영역에 업로드
  2. 2

    회원 데이터 확보 (Members List)

    대상 액션의 관리자 페이지로 이동하여 회원 관리 > 활동 회원 메뉴에서 회원 명단을 다운로드합니다.

    파일명 예시: members.xlsx Members List 영역에 업로드
  3. 3

    병합 및 결과물 생성

    두 파일이 모두 준비되면 상단의 데이터 병합 시작하기 버튼을 클릭합니다. 작업이 완료되면 결과 파일을 다운로드하여 하위 모임 사용자 이름이 포함된 데이터를 확인하실 수 있습니다.
